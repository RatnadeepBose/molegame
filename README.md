# Mole Game

## 1. Overview
Mole Game is an interactive Whack-a-Mole style game built with HTML, CSS, and JavaScript. It offers a fun, engaging experience for players of all ages, combining fast-paced gameplay with appealing visuals and sounds.

## 2. Features
The game includes multiple mole types with unique animations and sound effects. Its fully responsive design ensures smooth gameplay across desktops, tablets, and mobile devices. The lightweight structure allows quick loading and seamless performance.

## 3. Project Structure
The repository contains an `index.html` file for the main game interface, an `images` folder with all visual assets, and a `sound` folder containing the audio effects. The project is designed to be easily expandable, allowing users to add more mole boxes or levels.

## 4. How to Play
Players can open `index.html` locally in their browser or play online via the live demo: [Mole Game Live](https://ratnadeepbose.github.io/molegame/). The objective is to click on moles as they appear to score points, with each mole type providing a different challenge.
